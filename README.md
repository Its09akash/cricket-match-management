# cricket-match-management
Project Overview

This project is a PostgreSQL database for managing a cricket tournament.
The database stores information about teams, players, matches, and scorecards.

Features

- 4 Teams in the tournament
- Each team has 11 players
- Total 24 matches played
- Score tracking for each player

Database Tables

1. teams – Stores team information
2. players – Stores player details and team association
3. matches – Stores match details
4. scorecard – Stores player performance in each match

Example Queries

- Find the player with the highest total runs
- List players and their teams
- Match details and results
